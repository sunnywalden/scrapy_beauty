## 一、项目说明

通过scrapy框架，爬虫获取某图片站点中国美女、韩国美女等标签下的高清美女图片

## 三、部署

### 3.1 获取代码

```bash
git clone https://github.com/sunnywalden/scrapy_beauty.git
```
### 3.2 安装依赖

```bash
pip install -r requirements.txt
```

### 3.3 运行项目

```bash
scrapy crawl siwa
```
